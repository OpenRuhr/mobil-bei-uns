API & Open Source

Das Projekt mobil-bei-uns.de ist OpenSource Software. Es basiert auf dem Python-Web-Framework Flask und benötigt eine MySQL Datenbank. Die Quelldaten liegen auf Github, Patches sind willkommen.

Die Daten von mobil-bei-uns.de sind über eine offene Schnittstelle (API) abrufbar. Die Lizenz jedes einzelnen Datensatzes steht in den licence-Feldern.

Die Liste aller Verkehrsbehinderungen erhält man unter /construction-list, welches ein JSON zurückgibt:

[
  {
    'id': ID der Verkehrsbehinderung, Integer
    'lat': Geographische Breite, Float
    'lon': Geographische Länge, Float
  },
  ...
]

Mit der ID kann man dann weitere Informationen zu den einzelnen Verkehrsbehinderungen abfragen. Unter /construction-details?id=CONSTRUCTION_ID erhält man folgendes JSON

{
  'id': ID der Verkehrsbehinderung, Integer
  'descr': Beschreibung, String(35000)
  'lat': Geographische Breite, Float
  'lon': Geographische Länge, Float
  [...]
}
